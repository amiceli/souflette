# Souflette

A lightweight Electron app to quickly sort and filter images from a folder by keeping the good ones and deleting the unwanted ones. 

*Souflette* (a small air blower) is like dusting before cleaning.

## Installation

```bash
npm install
npm start
```

## Usage

1. Launch the app and select a folder with images
2. Navigate with arrow keys or buttons
3. Press **K** to keep or **D** to delete each image
4. Kept images are saved to your list

## Languages

Auto-detects French or English based on your system language.
